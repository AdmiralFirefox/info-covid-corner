A website that tracks all information regarding COVID-19. The data updates every day depending on the COVID API. 

[You can visit the live site here.](https://info-covid-corner.vercel.app/)

<br />

## Features
* Tracks confirmed cases, deaths and recovered by each country and globally.

* Can search for a country's confirmed cases, deaths and recovered.

* Also when searching for a country, it displays a line chart for tracking a country's confirmed cases and deaths in the past few days, as well as the new cases and new deaths every day.

* Can search for a country's administered, vaccinated and partially vaccinated population information.
